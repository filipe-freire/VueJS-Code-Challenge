# pluralo-challenge

## Project setup

```
pnpm install
touch .env.local
```

Inside the `.env.local`, declare a variable called `VUE_APP_TOKEN` and assign it to a valid authentication token.

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```
